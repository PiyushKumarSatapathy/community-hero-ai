# 🤖 Community Hero AI
![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white) ![Firebase](https://img.shields.io/badge/Firebase-Firestore-FFCA28?logo=firebase&logoColor=black) ![Google Gemini](https://img.shields.io/badge/Google-Gemini-4285F4?logo=google&logoColor=white) ![Google Maps](https://img.shields.io/badge/Google-Maps-34A853?logo=googlemaps&logoColor=white) ![Vite](https://img.shields.io/badge/Vite-7-646CFF?logo=vite&logoColor=white) ![Hackathon](https://img.shields.io/badge/Hackathon-Vibe2Ship%202026-purple) 

An AI-powered hyperlocal civic issue reporting platform built using **React**, **Firebase**, **Google Gemini AI**, and **Google Maps Platform**.

Community Hero enables citizens to report community issues, while AI automatically analyzes every report, assigns the responsible department, predicts severity, estimates resolution time, provides prevention tips, and generates actionable recommendations.

---

## ✨ Features

- 📍 Report community issues with Google Maps
- 🤖 AI-powered issue analysis using Google Gemini
- 🏷 Automatic category & department detection
- 🚦 Priority prediction (High / Medium / Low)
- 🔥 Severity assessment
- ⏳ Estimated resolution time
- 💡 AI-generated prevention tips
- 📊 Interactive analytics dashboard
- 🗺 Live community issue map
- ✔ Community verification system
- 🏆 Department performance leaderboard
- 📱 Fully responsive UI

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/PiyushKumarSatapathy/community-hero-ai.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
VITE_GOOGLE_MAPS_API_KEY=YOUR_API_KEY
```

Run the project

```bash
npm run dev
```

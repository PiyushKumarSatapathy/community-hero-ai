# Community Hero AI

Community Hero AI is an AI-powered hyperlocal civic issue reporting platform.

## Features

- Report community issues
- Dashboard analytics
- Citizen leaderboard
- AI-assisted issue categorization (planned)
- Firebase backend (planned)

## Tech Stack

- React
- Vite
- JavaScript
- Firebase (Upcoming)
- Gemini AI (Upcoming)
